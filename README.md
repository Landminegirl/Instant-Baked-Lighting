# here's a tool to bake your lights easy as hell cuz i got tired of using 4 separate tools to do it.

Click your active object, collection, or objects
set your resolution
click Create/Refresh Bake UV map (It'll make you a UV lightmap, don't worry about it)
select the channels you wanna bake
click "Bake Checked Channels" (It's gonna take a minute!)
click "create game export copy"
